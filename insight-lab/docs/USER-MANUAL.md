# InsightLab User Manual

A plain-language guide for students, teachers, and teams using InsightLab to study documents and spreadsheets with AI.

**In the app:** open **User guide** in the sidebar for the same help inside InsightLab. Use **Show tour** on any page for a quick walkthrough.

---

## What is InsightLab?

InsightLab helps you:

- Upload lecture notes, PDFs, Word files, and Excel spreadsheets
- Ask questions and get answers grounded in your files
- Generate quizzes, flashcards, study guides, and summaries
- Track progress and study with classmates on shared **study sheets**

Think of a **study sheet** as one folder for a class, exam, or project.

---

## Getting started

### 1. Sign in

Create an account or sign in with Google. You need an account to save files and share with others.

### 2. Create a study sheet

1. Go to **Study sheets** in the sidebar.
2. Type a name (for example, “Biology midterm”).
3. Click **Create**.

### 3. Upload files

1. Open your study sheet.
2. Click **Upload** (or **Upload materials**).
3. Choose PDF, Word, Excel, or CSV files.
4. Wait until status shows **Ready** (usually about a minute).

You need at least one **Ready** file before most study tools work.

## Roles: who can do what

| Role | Can view & study | Can upload & generate | Can share & manage members |
|------|------------------|------------------------|----------------------------|
| **Viewer** | Yes | No | No |
| **Editor** | Yes | Yes | Can invite (with owner) |
| **Owner** | Yes | Yes | Full control, can delete sheet |

## Study sheet page (your home base)

When you open a study sheet, you see several sections. Here is what each one is for.

### Progress

Quick stats: ready files, quiz scores, topics to review, and a **Study next** suggestion.

### Guided study plan (whole sheet)

This is your **step-by-step plan across every file** in the sheet.

1. Click **Start my plan** once to begin tracking progress.
2. Work through each step with **Go** (opens the right file or tool).
3. Click **Mark done** when you finish a step.
4. Use **Resume plan** to jump back to where you left off.

**Important:** You do not need to start the plan to use the app. You can open any file and use chat or quiz on your own. The plan is optional — it keeps you organized.

If you generated a **learning path** first, the plan follows that order.

### Learning path

AI suggests an order to study topics across your files. Generate a path, then start your guided plan to follow it.

### Concept graph (whole sheet)

A map of topics across all documents, colored by how well you know them. Filter to **weak concepts** to focus review.

### Team chat

Plain-text chat for sheet members. No links, files, or emoji — just short messages to coordinate with classmates. Updates appear live.

### Materials list

All uploaded files. Click a **Ready** file to open its workspace (chat, quiz, etc.).

### Practice quiz (whole sheet)

A quiz that pulls from **weak topics across all files**. You should take at least one quiz on a single file first so the app knows what you need to practice.

This is different from the guided study plan: the plan is a checklist; the practice quiz is the actual test.

### Compare documents

On the sheet page (or **Compare** in the sidebar), select multiple ready files and ask one question. You get a combined answer with citations.

### Source links (Excel sheets only)

If your study sheet includes **spreadsheets**, you can link each Excel file to a related PDF or Word document so spreadsheet chat can cite document excerpts. **PDF-only sheets do not need this** — open each file from Materials and use chat and quiz there. The Source links button appears only when the sheet has at least one spreadsheet.

### Course pack

Generates summary, quiz, flashcards, study guide, and audio for every ready document. Good for a fast bulk setup.

### Share

Invite classmates by email. Choose **viewer** (study only) or **editor** (can upload). Copy the invite link and send it yourself.

### Classroom analytics (editors)

See quiz averages and activity across members — useful for teachers.

## Document workspace (PDF / Word)

Open a ready document from your sheet.

### Chat

Ask questions about the file. Answers include citations you can click to see the source excerpt.

### Studio (right panel)

| Tab | What it does |
|-----|----------------|
| **Brief** | AI summary — read this first |
| **Study plan** | Guided brief → flashcards → quiz for *this file only* |
| **Quiz** | Practice questions; edit and publish before sharing |
| **Flashcards** | Flip cards; export to Anki CSV |
| **Study guide** | Structured outline; export to PDF |
| **Infographic** | Visual summary |
| **Audio overview** | Listen to a narrated summary |
| **Concept graph** | Topics from this file, colored by mastery |

### Guided study plan (single file)

Same idea as the sheet plan, but for one document:

1. **Start my plan** to track progress.
2. **Go** / **Mark done** on each step.
3. **Resume plan** to continue.

Quiz scores during a plan count toward your progress automatically.

## Excel workspace (spreadsheet)

Same notebook layout as documents.

| Tab | What it does |
|-----|----------------|
| **Chat** | Ask about trends, columns, and charts |
| **Insights** | AI summary of the data |
| **Preview** | Sample rows from the sheet |
| **Charts** | Suggested and saved charts; export CSV/PNG |
| **Builder** | Build custom charts from columns |
| **Concept graph** | Topics from insights and charts |
| **Quiz** | Questions based on the spreadsheet |

Use **Re-analyze** in the tools panel if you update the file.

## Compare (sidebar)

1. Choose a study sheet.
2. Select two or more ready documents.
3. Ask one question.
4. Read the combined answer with per-file citations.

Excel files open in the spreadsheet workspace; PDFs/Word open in the document workspace.

## Quizzes: two types (don’t mix them up)

| Name in the app | Scope | When to use |
|-----------------|-------|-------------|
| **Quiz** (on a file) | One document or spreadsheet | Learn one lecture or dataset |
| **Practice quiz (whole sheet)** | All files in the sheet | Review weak topics after file quizzes |

**Publish:** Editors can edit quiz questions, then **Publish** so others can take them. Draft quizzes are only visible to editors.

**Share publicly:** Published quizzes can get a share link for people outside the sheet (see quiz panel).

## Typical workflows

### Solo exam prep

1. Create a sheet → upload notes → wait for Ready.
2. Open each file → read **Brief** → take **Quiz**.
3. Start **Guided study plan** on the sheet to finish weak areas.
4. Use **Concept graph** to see what still needs work.

### Study group

1. Owner creates sheet and uploads materials.
2. **Share** invite links (viewer or editor).
3. Use **Team chat** to coordinate.
4. Everyone takes quizzes; **Practice quiz (whole sheet)** targets group weak spots.

### Teacher / classroom

1. Upload all course files to one sheet.
2. **Course pack** to generate artifacts.
3. **Classroom analytics** to monitor scores.
4. Export **Canvas** or **LMS bundle** from course pack drawer.

## Troubleshooting

| Problem | What to try |
|---------|-------------|
| File stuck processing | Wait a minute; click **Refresh** on the sheet |
| Feature unavailable (yellow notice) | Contact your administrator — the app may need a database update |
| Cannot upload | You may be a viewer — ask the sheet owner for editor access |
| Team chat not updating live | Messages still save; page refreshes automatically or on send |
| Weak topics empty | Take at least one quiz on a file first |

---

## Privacy & safety

- Only members of a study sheet can see its files and chat.
- Team chat is plain text only — no links or attachments.
- Do not upload confidential data you would not share with your study group.
